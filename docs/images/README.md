# Image Panels

Animated GIF and image sequence panels for LCDPossible

## Panels

| Panel | Description | Category |
|-------|-------------|----------|| [Animated GIF](panels/animated-gif/animated-gif.md) | Plays animated GIF files or URLs with full animation support | Media |
| [Image Sequence](panels/image-sequence/image-sequence.md) | Plays a sequence of numbered images from a folder at 30fps | Media |
