# Image Sequence

**Panel ID:** `image-sequence:`
**Category:** Media
**Plugin:** Image Panels
**Live Data:** Yes
**Animated:** Yes

Plays a sequence of numbered images from a folder at 30fps

## Overview

The LCDPossible panel provides seamless playback of animated sequences created from numbered image files, facilitating dynamic visual displays with alphanumerical organization and a steady frame rate ensuring consistent loop execution at thirty frames per second (fps). This functionality is particularly advantageous in media presentations requiring visually engaging content without the complexity of traditional animation software.

## Details

Plays a folder of numbered images as an animation.

File naming conventions supported:
- frame001.png, frame002.png, ...
- image_0001.jpg, image_0002.jpg, ...
- 001.png, 002.png, ...

Supported image formats: PNG, JPG, JPEG, BMP, GIF (static)

Images are sorted alphanumerically and played at 30fps.
The sequence loops continuously.

Tip: Export animations from video editing software as image sequences for smooth playback.

## Parameters

| Parameter | Description | Required | Default |
|-----------|-------------|----------|---------|
| `folder` | Path to folder containing numbered image files | Yes | - |

## Examples

### Play images from a folder as animation

```bash
lcdpossible show image-sequence:C:\frames\
```

### Play a relative folder path

```bash
lcdpossible show image-sequence:./export/frames/
```

### Play for 2 minutes then move to next panel

```bash
lcdpossible show image-sequence:C:\animation\|@duration=120
```


## Profile Usage

### Add to Profile

```bash
# Add panel to default profile
lcdpossible profile add image-sequence:

# Add with custom duration (30 seconds)
lcdpossible profile add "image-sequence:|@duration=30"
```

### Quick Show

```bash
# Display panel immediately
lcdpossible show image-sequence:
```
