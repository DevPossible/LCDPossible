# Animated GIF

**Panel ID:** `animated-gif:`
**Category:** Media
**Plugin:** Image Panels
**Live Data:** Yes
**Animated:** Yes

Plays animated GIF files or URLs with full animation support

## Details

Displays animated GIF files with automatic frame timing.

Supported sources:
- Local file paths (absolute or relative)
- HTTP/HTTPS URLs
- Network shares (Windows UNC paths)

The GIF is automatically scaled to fit the display while maintaining aspect ratio.
Frame timing is preserved from the original GIF.

Tip: Use high-quality GIFs at or near your LCD resolution for best results.

## Parameters

| Parameter | Description | Required | Default |
|-----------|-------------|----------|---------|
| `path` | Path to the GIF file or URL | Yes | - |

## Examples

### Play a local GIF file

```bash
lcdpossible show animated-gif:C:\gifs\animation.gif
```

### Play a GIF from a URL

```bash
lcdpossible show animated-gif:https://upload.wikimedia.org/wikipedia/commons/2/2c/Rotating_earth_%28large%29.gif
```

### Play a GIF for 60 seconds then advance to next panel

```bash
lcdpossible show animated-gif:./logo.gif|@duration=60
```


## Profile Usage

### Add to Profile

```bash
# Add panel to default profile
lcdpossible profile append-panel animated-gif:

# Add with custom duration (30 seconds)
lcdpossible profile append-panel "animated-gif:|@duration=30"
```

### Quick Show

```bash
# Display panel immediately
lcdpossible show animated-gif:
```
