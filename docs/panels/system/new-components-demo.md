# New Components Demo

**Panel ID:** `new-components-demo`
**Category:** Demo
**Plugin:** LCDPossible Core Panels
**Live Data:** Yes
**Animated:** No

Demo panel showcasing ECharts and DaisyUI components

## Details

Demonstrates the new graphical components:
- ECharts gauges (arc style)
- DaisyUI radial-progress gauges
- ECharts donut charts
- DaisyUI donut charts
- ECharts sparklines
- Both ECharts and DaisyUI progress bars

## Dependencies

- LibreHardwareMonitorLib
- PuppeteerSharp
