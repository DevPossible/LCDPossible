# Web Panels

HTML and web page panels using headless browser rendering (PuppeteerSharp).

## Quick Reference

```bash
# Render local HTML file
lcdpossible show html:path/to/dashboard.html

# Render live website
lcdpossible show web:https://wttr.in/London
```

## Panels

| Panel | Description | Category |
|-------|-------------|----------|
| [HTML File](panels/html/html.md) | Renders a local HTML file using headless browser | Web |
| [Web Page](panels/web/web.md) | Renders a live website from URL | Web |
