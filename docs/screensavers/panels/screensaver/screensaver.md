# Random Screensaver

**Panel ID:** `screensaver:`
**Category:** Screensaver
**Plugin:** Screensaver Panels
**Live Data:** Yes
**Animated:** Yes

Plays a random screensaver effect or cycles through all

## Overview

The Screensaver panel within LCDPossible offers users an array of visual experiences by facilitating access to various predefined screensavers that can be invoked through three distinct modes: a random selection mode for unpredictability and novelty; a cycle mode allowing seamless transitions between all available effects, thereby ensuring continuous engagement without repetition fatigue; and an individualized approach whereby users may select specific screensaver designs by name. These capabilities cater to both personal preference in visual stimulation during idle periods as well as the practical need for dynamic user interface elements within enterprise environments that seek creative displays or decorations using professional-grade technology.

## Details

Meta-panel that plays screensaver effects.

Modes:
- screensaver:random - Pick a random screensaver
- screensaver:cycle - Cycle through all screensavers
- screensaver:<name> - Play a specific screensaver

Available screensavers: starfield, matrix-rain, bouncing-logo, mystify, plasma, fire, game-of-life, bubbles, rain, spiral, clock, noise, warp-tunnel, pipes, asteroids, missile-command, falling-blocks

## Parameters

| Parameter | Description | Required | Default |
|-----------|-------------|----------|---------|
| `mode` | Screensaver mode or specific effect name | No | `random` |

## Examples

### Play a random screensaver

```bash
lcdpossible show screensaver:random
```

### Cycle screensavers every 30 seconds

```bash
lcdpossible show screensaver:cycle|@duration=30
```


## Profile Usage

### Add to Profile

```bash
# Add panel to default profile
lcdpossible profile add screensaver:

# Add with custom duration (30 seconds)
lcdpossible profile add "screensaver:|@duration=30"
```

### Quick Show

```bash
# Display panel immediately
lcdpossible show screensaver:
```
