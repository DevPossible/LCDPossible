# Video

**Panel ID:** `video:`
**Category:** Media
**Plugin:** Video Panels
**Live Data:** Yes
**Animated:** Yes

Plays video files, streaming URLs, or YouTube links

## Overview

The LCDPossible application is designed as an advanced video playback solution catering to professionals seeking seamless media presentation capabilities within their enterprise environments. Leveraging robust support for various local and online video sources alongside features like hardware acceleration, automatic aspect ratio adjustment, audio mute-by-default functionality with looping options, it is ideal for dynamic presentations where high performance and user convenience are paramount. The application's adeptness in handling YouTube content by autoextraction of suitable streams further broadens its utility to incorporate web video playback directly into workflow processes without necessitating additional software installations.

## Details

Full video playback panel powered by LibVLC.

Supported sources:
- Local video files (MP4, MKV, AVI, MOV, WebM, etc.)
- Direct video URLs (HTTP/HTTPS streams)
- YouTube video links (auto-extracts stream URL)

Features:
- Hardware-accelerated decoding when available
- Automatic aspect ratio handling
- Audio is muted by default
- Loops continuously

YouTube support:
- Full YouTube URLs: https://www.youtube.com/watch?v=VIDEO_ID
- Short URLs: https://youtu.be/VIDEO_ID
- Extracts highest quality stream up to 720p for performance

Note: Video playback requires LibVLC native libraries.

## Examples

### Play a local video file

```bash
lcdpossible show video:C:\videos\demo.mp4
```

### Stream a video from Archive.org (CC-BY)

```bash
lcdpossible show video:https://archive.org/download/BigBuckBunny_124/Content/big_buck_bunny_720p_surround.mp4
```

### Play a YouTube video (Big Buck Bunny, CC-BY)

```bash
lcdpossible show video:https://www.youtube.com/watch?v=aqz-KE-bpKQ
```

### Play video for 30 seconds then advance

```bash
lcdpossible show video:./intro.mp4|@duration=30
```

### Combine system info with video in a slideshow

```bash
lcdpossible show basic-info,video:./background.mp4
```


## Profile Usage

### Add to Profile

```bash
# Add panel to default profile
lcdpossible profile append-panel video:

# Add with custom duration (30 seconds)
lcdpossible profile append-panel "video:|@duration=30"
```

### Quick Show

```bash
# Display panel immediately
lcdpossible show video:
```
